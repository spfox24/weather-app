# Weather Application

## Weather or Not

A weather application to source current weather data by City.

## Version 1.0
Current build searches weather by City, and returns:
- City
- Temp
- Index
- Weather Description
- Display Icon of Weather Description

### Technologies
- HTML5
- CSS3
- JavaScript
- jQuery v3.5
- Open Weather API
